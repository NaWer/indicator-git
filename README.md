# indicator-git
A simple indicator with notification for git repositories

WORK IN PROGRESS

## Features
* Works on all git repositories (including bare)
* Mirrors the original repository
* Notification :
	* Commits
	* Branches
	* Tags

## Depends
 * python-appindicator
